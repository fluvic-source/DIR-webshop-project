Barrierefreiheit

Wissensraum legt Wert auf eine zugängliche Nutzung der Website. Die Oberfläche wurde mit Fokus auf klare Struktur, Tastaturbedienbarkeit, sichtbare Fokusmarkierungen und gute Lesbarkeit umgesetzt. Diese Website wurde mit Blick auf WCAG 2.2 AA entwickelt. Die Website wurde unter anderem mit automatisierten Accessibility-Tools wie WAVE überprüft. Dabei wurden keine kritischen Fehler oder Kontrastprobleme festgestellt. Eine vollständige technische Prüfung ist erforderlich, um die Konformität abschließend zu bestätigen.

  • Semantisches HTML
  • Tastaturbedienbarkeit
  • Ausreichende Farbkontraste
  • Responsive Darstellung
  • Klare Navigationsstruktur
  • Sichtbare Fokus-Markierungen
